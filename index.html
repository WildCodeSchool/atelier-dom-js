Atelier DOM JS

Objectif : dynamisez une page web HTML simple

Structure du projet

• Créez un nouveau fichier « index.html » avec le code suivant : 

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <title>
      Document
    </title>
  </head>
  <body>
    <button id="add">
      Add row
    </button>
    <button id="del">
      Remove row
    </button>
    <button id="hide">
      Hide
    </button>
    <button id="show">
      Show
    </button>
    <div>
      <ul>
        <li>Text 1</li>
        <li>Text 2</li>
        <li>Text 3</li>
        <li>Text 4</li>
      </ul>
    </div>
  </body>
</html>

• Créez un fichier « script.js »

Dans votre fichier HTML

Liez votre fichier JavaScript à votre fichier HTML.

Astuce : utilisez la balise script selon la bonne pratique évoquée en cours.

Dans votre fichier JavaScript

  • Créez un écouteur d'événement sur votre bouton qui porte l’id « add ».

  • Au clic sur ce bouton, créez un nouvel élément de liste en JavaScript et insérez-le à la fin de la balise « ul ».

    Astuce : regardez du côté de createElement et insertAdjacentElement.

  • Créez un écouteur d'événement sur votre bouton qui porte l’id « del ».

  • Au clic sur ce bouton, sélectionnez le dernier élément de la liste et supprimez-le.

    Astuce : utilisez le pseudo-selecteur CSS last-child pour la sélection.

  • Créez un écouteur d'événement sur votre bouton qui porte l’id « hide ».

  • Au clic sur ce bouton, cachez la liste en lui appliquant une classe CSS.

    Astuce : utilisez classList.

  • Créez un écouteur d'événement sur votre bouton qui porte l’id « show »

  • Au clic sur ce bouton, ré-affichez la liste en supprimant la classe CSS précédemment appliquée

    Astuce : utilisez classList.

Enjoy
